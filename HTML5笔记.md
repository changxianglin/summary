### HTML5 笔记
    * 中文网页使用 <meta charset="utf-8"> 否则会出现乱码
    * 新元素
    * 新属性
    * 完全支持 CSS3
    * Video 和 Audio
    * 2D/3D 制图
    * 本地存储
    * 本地 SQL 数据
    * Web 应用
    * HTML5 新加的块标签，header, section, footer, aside, nav, main, article, figure
    * HTML5 可自定义标签只要封闭加可用
    * canvas 主要和 js 一起运用才能出很好的效果
    * SVG 是主要在使用 XML 时运用
    * HTML5 的 input 新类型 color、 date、 datetime、 datetime-local、 email、 mont、 number、 range、 search、 tel、 tim、 url、 week
